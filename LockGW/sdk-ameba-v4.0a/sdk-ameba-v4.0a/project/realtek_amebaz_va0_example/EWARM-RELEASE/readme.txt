工程说明：
  测试成功日期：2018-01-17
  功能：
    1、绑定“信驰达科技”阿里云后台成功，设备只能绑定一个账号，新账号绑定后以前设备自动解绑；
    2、APP“ONEAPK”可配置操作成功；
    3、命令“ATCA=k,xxxxxxx”设置deviceID，20字节长度，同时设置后会擦除原来flash中保存的ID和密匙值；
    4、命令“ATCA=s,xxxxxxx”设置deviceSecret，40字节长度，注意此命令必须在先设置deviceID后才能写入成功；
    5、按键PA_0是激活设备配置的按键，在APP提示激活时短按按键可激活设备；
    6、命令“ATCA=e”，擦除原有的WiFi配置信息，并且设备重启需要重新配置设备联网和绑定(若账号本身绑定过设备且未解绑只需配置联网即可)；
    7、命令“ATCA=r”，解除设备绑定信息，WiFi配置信息不变，
    8、APP端可以主动解除绑定；
    9、默认的上报错误的最小间隔是2秒，修改成200ms；
    10、新增按键长按5秒擦除WiFi配置信息，并且设备重启。
